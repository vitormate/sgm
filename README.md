# Sistema de Gerenciamento de Máquinas (SGM)

O Sistema de Gerenciamento de Máquinas (SGM) é uma solução voltada para o setor industrial, desenvolvida para acompanhar e registrar manutenções de máquinas que utilizam horímetro (como empilhadeiras, transpaleteiras, entre outras).

O projeto foi motivado pelo alto desgaste dessas máquinas e as sucessivas idas para manuteção. O SGM tem como objetivo fornecer informações confiáveis para os líderes, permitindo consultas ao histórico de manutenção de cada máquina.

## Tecnologias Utilizadas

- **Java / Spring Boot** - *Backend*

- **PostgreSQL** - *Banco de dados*

### Principais Bibliotecas

- **Spring Data JPA** - *Integração e mapeamento com o banco de dados*

- **Spring Validation** - *Validação de dados de entrada*

- **Auth0** - *Autenticação e controle de acesso*
